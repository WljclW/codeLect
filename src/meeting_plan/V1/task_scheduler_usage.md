# 任务轮次调度器 - 完整使用手册

## 一、这是什么

这是一个排程工具。你告诉它"我有多少天、要做几轮任务"，它会自动判断能不能排得下，能的话生成一份 Excel 表格告诉你每天该干什么。

每一轮任务包含 A/B/C 三个环节的会议：
- A 环节：轮次第 1 天开会（1天）
- B 环节：轮次第 11、12 天开会（2天）
- C 环节：轮次第 15 天开会（1天）

核心约束：不同轮的会议不能撞在同一天。非会议日可以并行做别的事。

---

## 二、环境准备（第一次使用需要做，以后不用重复）

### 第1步：安装 Python

1. 打开浏览器，访问 Python 官网下载页面：
   ```
   https://www.python.org/downloads/
   ```
2. 点击页面上方黄色的 **"Download Python 3.x.x"** 按钮，下载安装包
3. 双击下载好的安装包（文件名类似 `python-3.x.x-amd64.exe`）
4. **非常重要**：在安装界面底部，勾选 **"Add Python to PATH"**（把Python加入环境变量）
5. 点击 **"Install Now"** 等待安装完成
6. 安装完成后点击 **"Close"**

#### 验证 Python 是否安装成功

1. 按键盘 `Win + R`，弹出"运行"对话框
2. 输入 `cmd`，按回车，打开黑色的命令提示符窗口
3. 输入以下命令并按回车：
   ```
   python --version
   ```
4. 如果显示类似 `Python 3.12.x` 的版本号，说明安装成功
5. 如果提示"不是内部或外部命令"，说明安装时没有勾选 "Add to PATH"，请卸载后重新安装并勾选

### 第2步：安装依赖库

1. 保持上一步的命令提示符窗口打开（如果关了就重新按 `Win + R` → 输入 `cmd` → 回车）
2. 输入以下命令并按回车：
   ```
   pip install openpyxl
   ```
3. 等待几秒钟，看到 `Successfully installed openpyxl-x.x.x` 字样即安装成功
4. 如果提示 `pip 不是内部或外部命令`，尝试输入：
   ```
   python -m pip install openpyxl
   ```

### 第3步：将程序文件放到电脑上

确保以下文件保存在你电脑的某个文件夹中（比如桌面或文档）：

- `task_scheduler_gui.py` — 这是程序本体

如果文件是别人发给你的，直接复制到你想放的位置即可。

## 三、运行程序

### 方法一：双击运行（推荐）

1. 在文件资源管理器中找到 `task_scheduler_gui.py`
2. **双击**该文件
3. 程序窗口弹出，进入下一节"操作步骤"

### 方法二：如果双击没反应

1. **右键** `task_scheduler_gui.py` → 选择 **"打开方式"** → **"选择其他应用"**
2. 在列表中找到 **Python** 并选中
3. 勾选 **"始终使用此应用打开 .py 文件"**
4. 点击确定

### 方法三：通过命令行运行

1. 在文件所在的文件夹空白处，按住 `Shift` 键并**右键**空白区域
2. 选择 **"在此处打开 PowerShell 窗口"**（或"在此处打开命令窗口"）
3. 输入：
   ```
   python task_scheduler_gui.py
   ```
4. 按回车，程序窗口弹出

## 四、操作步骤

程序启动后你会看到如下窗口界面：

```
┌──────────────────────────────────┐
│       任务轮次调度器              │
│                                  │
│  规则：每轮含A/B/C三个环节       │
│    A环节：轮次第1天开会          │
│    B环节：轮次第11、12天开会     │
│    C环节：轮次第15天开会         │
│  约束：不同轮的会议日不能同天    │
│                                  │
│  总天数：[        ]              │
│  轮  数：[        ]              │
│                                  │
│       [ 开始排程 ]               │
│                                  │
└──────────────────────────────────┘
```

### 步骤1：填写总天数

在"总天数"输入框中输入你拥有的总工作天数。例如输入 `30`。

### 步骤2：填写轮数

在"轮数"输入框中输入你要完成的任务轮数。例如输入 `3`。

### 步骤3：点击"开始排程"

点击蓝色的"开始排程"按钮。程序会立即计算结果。

### 步骤4：查看结果

#### 情况A — 可以实现

1. 弹出"保存文件"对话框，让你选择 Excel 文件的保存位置
2. 选好位置后点击"保存"
3. 弹出成功提示，显示具体的排程安排
4. 点击"是"可以立即打开生成的 Excel 文件

#### 情况B — 无法实现

弹出提示框告知原因（天数不够），你需要增加总天数或减少轮数后重试。

## 五、生成的 Excel 文件说明

打开 Excel 后你会看到一张表格：

| 天数 | 第1轮 | 第2轮 | 第3轮 | 备注 |
|------|-------|-------|-------|------|
| 第1天 | **A会议（红色）** | | | 第1轮A环节会议 |
| 第2天 | | | | 可并行其他工作 |
| 第3天 | | **A会议（红色）** | | 第2轮A环节会议 |
| ... | ... | ... | ... | ... |
| 第11天 | **B会议（橙色）** | | | 第1轮B环节会议 |
| 第12天 | **B会议（橙色）** | | | 第1轮B环节会议 |
| 第13天 | | **B会议（橙色）** | | 第2轮B环节会议 |
| ... | ... | ... | ... | ... |

**颜色含义：**
- 红色格子 = A环节会议日
- 橙色格子 = B环节会议日
- 绿色格子 = C环节会议日
- 空白格子 = 该天无会议，可以并行做其他工作

**表格底部**有汇总信息，列出每一轮的起始日和各环节具体日期。

## 六、常见问题

| 问题 | 解决办法 |
|------|----------|
| 双击 .py 文件后弹出记事本 | 右键 → 打开方式 → 选择 Python（参考第三节方法二） |
| 提示"python 不是内部或外部命令" | Python 安装时未勾选 Add to PATH，卸载重装并勾选 |
| 提示"No module named 'openpyxl'" | 打开 cmd，执行 `pip install openpyxl` |
| 提示"No module named 'tkinter'" | tkinter 随 Python 自带，重新安装 Python 时勾选所有默认组件 |
| 输入后提示"无法实现" | 天数不够，增大总天数或减少轮数 |
| Excel 打开是乱码 | 确保用 Microsoft Excel 或 WPS 打开 .xlsx 文件 |
| 生成的文件找不到了 | 下次保存时记住选择的路径，建议保存到桌面 |

## 七、完整示例演示

假设你有 **30天**，需要完成 **3轮**任务：

1. 双击 `task_scheduler_gui.py`
2. 总天数输入 `30`
3. 轮数输入 `3`
4. 点击"开始排程"
5. 选择保存到桌面，文件名保持 `task_schedule.xlsx`
6. 点击保存
7. 弹窗显示结果：
   ```
   第1轮：A=第1天, B=第11&12天, C=第15天
   第2轮：A=第3天, B=第13&14天, C=第17天
   第3轮：A=第8天, B=第18&19天, C=第22天
   ```
8. 点击"是"打开 Excel 查看完整日历安排

## 八、技术信息（可忽略）

- 运行环境：Python 3.8 及以上
- 依赖库：openpyxl（生成Excel）、tkinter（图形界面，Python自带）
- 算法：贪心调度，为每轮寻找最早的不冲突起始日
- 系统要求：Windows 7/10/11
